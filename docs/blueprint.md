# **App Name**: Image tag generator

## Core Features:

- Image Upload: Drag-and-drop interface for image uploading.
- AI Alt Text Generation: Generative AI tool that analyzes uploaded images and suggests descriptive 'alt' text, suitable for accessibility purposes.
- HTML Tag Generation: Generates and displays the HTML image tag, including src, width, height, and AI-generated alt text.
- Copy to Clipboard: A copy icon to copy the HTML image tag to the clipboard.

## Style Guidelines:

- Primary color: Sky blue (#63B5FF), offering a friendly, digital feel.
- Background color: Light, desaturated blue (#E5F1FF), providing a subtle contrast and calm background.
- Accent color: Light green (#90EE90) used to indicate completion and confirmation for copy operations or successful image analysis.
- Clean, sans-serif font to ensure readability.
- Simple, recognizable icons for actions like 'copy' and 'upload.'
- Clean layout with clear visual hierarchy, emphasizing ease of use.
- Subtle transitions and feedback animations upon image upload, analysis, and code generation.
